# PathFinder

## How to Run
python search.py OR py search.py

## Requirements
- Python 3.11
- matplotlib

## Testing Grid
- If you'd like to change which world to test, locate "__main__" and switch: world1 -> world2, or vice versa

## Algorithms
- BFS: Breadth-First Search using FIFO (Stack)
- DFS: Depth-First Search using LIFO (Queue)
- GBSF: Greedy Best-First Search using SLD (Straight Line Distance) Heuristic
- A*: A* Search using g(n) + h(n) with SLD Heuristic
